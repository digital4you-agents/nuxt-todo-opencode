# Nuxt Todo (OpenCode Test)

Kleine **Todo-App** mit **Nuxt**.

## Features
- Todos hinzufügen / erledigen / löschen
- Erledigte Todos optional ausblenden
- Persistenz via `localStorage`

## Screenshot

![Better Screenshot](docs/new-screenshot.png)

## Setup

```bash
npm install
```

## Dev

```bash
npm run dev
```

Open: http://localhost:3000

## Build

```bash
npm run build
npm run preview
```
